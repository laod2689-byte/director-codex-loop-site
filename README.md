# Director-Codex Loop Site

This is a standalone public page for the `director-codex-loop` project.

## Local preview

```powershell
cd C:\Users\ZhuanZ\director-codex-loop-site
python -m http.server 8000
```

Open `http://localhost:8000/`.

## GitHub Pages

1. Create a new public GitHub repository.
2. Push this directory as that repository.
3. In GitHub repository settings, enable Pages from the `main` branch and `/(root)`.

The public URL will be similar to:

```text
https://<your-github-username>.github.io/<repo-name>/
```
